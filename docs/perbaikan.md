## ğŸ¯ Phase 1: Restructure HeroSection (Biggest Goal)

### Masalah Saat Ini:
- Judul terlalu generic: "Rahasia Memilih Jurnal Scopus yang Tepat"
- Tidak fokus ke hasil akhir yang diinginkan user
- Tidak menggunakan format "[Bantu] [siapa] [mencapai apa]"
- Tagline dari poster tidak digunakan: "Hindari Artikel Bagus Ditolak Karena Salah Pilih Jurnal"

### Saran Perbaikan:

#### 1.1 Ubah Headline HeroSection

**Dari:**
```
Rahasia Memilih Jurnal Scopus yang Tepat untuk Publikasi Anda
```

**Menjadi (sesuai Framework 2):**
```
Bantu Dosen & Peneliti Publikasi di Jurnal Scopus Berkualitas Tanpa Takut Ditolak atau Terjebak Jurnal Predatory
```

**Atau alternatif:**
```
Bantu Anda Hindari Artikel Bagus Ditolak Karena Salah Pilih Jurnal Scopus
```

**Penjelasan:**
- Format: [Bantu] [siapa] [mencapai apa]
- Fokus ke hasil akhir: "Publikasi di Scopus" dan "Hindari ditolak"
- Menggunakan tagline dari poster

#### 1.2 Perbaiki Sub-headline

**Dari:**
```
Putus asa manuskrip Anda tak kunjung tembus Scopus? Kami bahas tuntas di webinar kami ini.
```

**Menjadi:**
```
Saya bantu Anda kuasai strategi memilih jurnal Scopus yang tepat, sehingga artikel berkualitas Anda tidak lagi ditolak hanya karena salah memilih jurnal target.
```

**Penjelasan:**
- Menggunakan "Saya" (personal approach)
- Fokus ke hasil: "tidak lagi ditolak"
- Lebih spesifik dan relatable

#### 1.3 Tambahkan Hook Statement

**Tambahkan setelah sub-headline:**
```
Bayangkan 3 bulan ke depan, artikel Anda sudah diterima di jurnal Scopus Q2 atau Q3 yang tepat, tanpa harus trial-error berkali-kali dan menghabiskan waktu berbulan-bulan.
```
